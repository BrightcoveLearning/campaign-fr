Conduite des vues vidéo par e-mail

{{ page.description }}

La fonction Publier dans Brightcove Campaign™ peut être utilisé pour générer facilement le code d'intégration et les images nécessaires pour distribuer une vidéo à l'aide d'une page de destination et d'un e-mail. Généralement, le code d'intégration vidéo est collé dans une page de destination, puis un lien vers la page de destination est envoyé par e-mail. Le processus décrit dans cette rubrique est le mieux adapté lorsqu'une plateforme d'automatisation du marketing (MAP) est ne pas utilisé. Si vous utilisez un MAP, Brightcove recommande d'utiliser le processus de publication.

Création d'une page de destination

Une page de destination est une page Web autonome qui peut être utilisée dans le cadre d'une campagne marketing pour promouvoir un produit ou un service. L'ajout de vidéos aux pages de destination est un excellent moyen d'accroître l'engagement des spectateurs et de transformer les spectateurs en acheteurs. Il existe un certain nombre de fournisseurs qui offrent la possibilité de créer des pages de destination. Les pages de destination peuvent également être créées à l'aide d'un logiciel de plateforme d'automatisation marketing ou manuellement, en utilisant HTML personnalisé si vous disposez de ressources de développeur. Une fois que vous avez créé une page de destination, Brightcove Campaign peut être utilisé pour incorporer une vidéo dans la page de destination.

Génération du code d'intégration

Le code d'intégration est requis pour qu'une vidéo soit lue sur une page de destination. Pour copier le code intégré d'une vidéo, procédez comme suit :

  1. Ouvrez la page Vidéos .
  2. Sélectionnez une vidéo en cochant la case située en regard de la vidéo que vous souhaitez utiliser.
  3. Cliquez sur Publish.
  4. Copiez le Code intégré et collez-le dans votre page de destination à l'endroit où vous souhaitez que la vidéo apparaisse.
  5. Enregistrez les modifications apportées à votre page de destination.
  6. Publiez la page de destination et assurez-vous que vous disposez d'une URL sur laquelle la page de destination peut être affichée.

Copie de l'image de l'affiche

Sur la plupart des plateformes, il n'est pas’possible d'intégrer des vidéos directement dans les e-mails. Il ne s'agit pas d'une restriction de Brightcove. Pour des raisons de sécurité, presque tous les systèmes de messagerie modernes suppriment le code d'intégration au lecteur vidéo. Nous pouvons cependant stimuler la sensibilisation et le trafic pour le contenu en créant un e-mail avec du texte teaser et une image d'affiche. L'image de l'affiche ressemblera à un lecteur vidéo et sera un lien cliquable qui redirige vers la page de destination.

Pour copier l'image de l'affiche d'une vidéo, procédez comme suit :

  1. Clique le E-mail onglet en haut de la page.
  2. Cliquez sur Copier pour copier l'image de l'affiche.
  3. Collez l'image dans l'outil qui va envoyer l'e-mail.
  4. Créez un lien hypertexte entre l'image et la page de destination. Le processus pour ce faire varie en fonction du programme de courriel utilisé.

Lorsque l'e-mail est envoyé, les lecteurs verront la vidéo dans le courriel et lorsqu'ils cliquent sur l'image, ils seront redirigés vers la page de destination où ils peuvent regarder la vidéo.
