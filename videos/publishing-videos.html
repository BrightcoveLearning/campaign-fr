Publication de vidéos

{{ page.description }}

La fonction Publier dans Brightcove Campaign™ peut être utilisé pour générer facilement le code d'intégration et les images nécessaires à la diffusion d'une vidéo à l'aide d'une plate-forme d'automatisation du marketing (MAP). En règle générale, le code intégré est collé dans une page de destination HTML créée dans un MAP, puis un lien vers la page de destination est envoyé par e-mail.

Publication sur une page Web

Pour copier le code d'intégration d'une vidéo, procédez comme suit :

  1. Ouvrez la page Vidéos .
  2. Sélectionnez une vidéo en cochant la case située en regard de la vidéo.
  3. Cliquez sur Publish.
  4. (Facultatif) Sélectionnez un Connexion MAP. La liste déroulante contiendra une liste de tous Connexions MAP qui ont été créés. En sélectionnant une connexion MAP, l'activité de visualisation vidéo est envoyée à la plateforme sélectionnée. La disponibilité de cette option est basée sur le package Brightcove Campaign utilisé.
  5. (Facultatif) Sélectionnez un Formulaire de plomb. La liste déroulante contiendra une liste de tous formulaires de plomb qui ont été créés. En sélectionnant une connexion MAP, un formulaire prospect peut être affiché pendant la lecture vidéo pour capturer les informations du spectateur. La disponibilité de cette option est basée sur le package Brightcove Campaign utilisé.
  6. Copiez le code intégré et collez-le dans votre page de destination. C'est le code qui est nécessaire pour publier votre vidéo sur une page web.
  7. Copiez le Code de référencement vidéo et collez le code entre <head> et </head> balises sur votre page de destination. Le code SEO vidéo aidera à améliorer la visibilité de la page de destination pour les moteurs de recherche. Une fois votre page publiée, Google fournit un Outil d'inspection d'URL qui peut être utilisé pour afficher des informations détaillées de l'index Google sur une URL de votre site.
  8. Copiez le Tags Facebook et collez le code entre <head> et </head> balises sur votre page de destination. Les balises Facebook sont les balises Facebook Open Graph qui aideront à contrôler ce qui apparaît lorsque la page est partagée sur Facebook.

Copie de l'image de l'affiche

Sur la plupart des plateformes, il n'est pas’possible d'intégrer des vidéos directement dans les e-mails. Il ne s'agit pas d'une restriction de Brightcove. Pour des raisons de sécurité, presque tous les systèmes de messagerie modernes suppriment le code d'intégration au lecteur vidéo. Nous pouvons cependant stimuler la notoriété et le trafic du contenu en créant un e-mail contenant du texte de teaser et une image d'affiche. L'image de l'affiche ressemblera à un lecteur vidéo et sera un lien cliquable qui redirige vers la page de destination de la vidéo.

Pour copier l'image de l'affiche d'une vidéo, procédez comme suit :

  1. Clique le E-mail onglet en haut de la page.
  2. Cliquez sur Copier pour copier l'image de l'affiche.
  3. Collez l'image dans l'outil qui va envoyer l'e-mail.
  4. Créez un lien hypertexte de l'image vers la page de destination avec la vidéo publiée.
