Téléchargement et publication de votre première vidéo

{{ page.description }}

Campagne de Brightcove™ facilite le téléchargement de vidéos, puis la génération du code d'intégration et des images nécessaires pour diffuser une vidéo à l'aide d'une plate-forme d'automatisation du marketing (MAP). En règle générale, le code intégré est collé dans une page de destination HTML créée dans un MAP, puis un lien vers la page de destination est envoyé par e-mail.

Chargement de vidéos

Brightcove Campaign facilite le téléchargement de vos vidéos. Pour plus d'informations sur le téléchargement de vidéos, consultez Téléchargement de vidéos.

Pour télécharger des vidéos sur votre compte, procédez comme suit.

  1. Cliquez sur Vidéos dans l'en-tête de navigation.
  2. Cliquez sur Charger. Le panneau de téléchargement s'ouvre dans la page.
  3. Cliquez sur Rechercher des vidéos et parcourez et sélectionnez une vidéo à télécharger.
  4. Une fois les fichiers sélectionnés, la progression du téléchargement s'affiche. Cliquez sur Modifier pour afficher les propriétés de la vidéo.

Une fois le téléchargement terminé, un Téléchargé l'état s'affichera. Cela indique que le téléchargement de la vidéo est terminé et que vous pouvez vous éloigner du Vidéos page.

Publication d'une vidéo

Le processus de publication générera le code d'intégration HTML nécessaire pour afficher votre vidéo sur une page Web. Pour accèder/copier le code d'intégration d'une vidéo, procédez comme suit :

  1. Ouvrez le Vidéos page.
  2. Sélectionnez une vidéo en cochant la case à côté de la vidéo que vous souhaitez publier.
  3. Cliquez sur Publish.
  4. Sélectionner un Connexion MAP. La liste déroulante contient une liste de toutes les connexions MAP qui ont été créées. En sélectionnant une connexion MAP, l'activité de visualisation vidéo est envoyée à la plateforme sélectionnée.
  5. Copiez le code intégré et collez-le dans votre page de destination.
  6. Copiez le Code de référencement vidéo et collez-le entre le <head> et </head> balises sur votre page de destination. Le code SEO vidéo aidera à améliorer la visibilité de la page aux moteurs de recherche.
  7. Copiez le Balises Facebook et collez-le entre le <head> et </head> balises sur votre page de destination. Les balises Facebook sont les balises Facebook Open Graph qui aideront à contrôler ce qui apparaît lorsque la page est partagée sur Facebook.

Copie de l'image de l'affiche

Sur la plupart des plateformes, il n'est pas’possible d'intégrer des vidéos directement dans les e-mails. Il ne s'agit pas d'une restriction de Brightcove. Pour des raisons de sécurité, presque tous les systèmes de messagerie modernes suppriment le code d'intégration au lecteur vidéo. Nous pouvons cependant stimuler la notoriété et le trafic du contenu en créant un e-mail contenant du texte de teaser et une image d'affiche. L'image de l'affiche ressemblera à un lecteur vidéo et sera un lien cliquable qui redirige vers la page de destination de la vidéo.

Pour copier l'image de l'affiche d'une vidéo, procédez comme suit :

  1. Clique le E-mail onglet en haut de la page.
  2. Cliquez sur Copier pour copier l'image de l'affiche.
  3. Collez l'image dans l'outil qui va envoyer l'e-mail.
  4. Créez un lien hypertexte entre l'image et la page de destination.
