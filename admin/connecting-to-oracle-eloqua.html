Connexion à Oracle Eloqua

{{ page.description }}

Brightcove Campaign peut être configuré pour s'intégrer à Oracle Eloqua. Lorsque l'intégration est configurée, les données d'engagement vidéo pour les vidéos publiées à l'aide de Brightcove Campaign sont synchronisées avec Eloqua. Une fois dans Eloqua, ces données peuvent être utilisées pour la segmentation, la logique de campagne, les rapports, la notation des prospects et la personnalisation des communications.

Exigences

Pour utiliser Brightcove Campaign avec Oracle Eloqua, vous devez :

  • Disposez d'Oracle Eloqua 10 et de vos identifiants pour vous connecter à : https://login.eloqua.com
    Noter: Ces informations d'identification sont saisies dans une page de connexion Eloqua et ne sont pas fournies directement à Brightcove.
  • Les informations d'identification utilisées pour se connecter à Eloqua doivent être ajoutées au Utilisateurs avancés - Marketing groupe et disposez des autorisations suivantes:
    • Consommer l'API
    • Installer des applications cloud
    • Concetter des campagnes marketing

Fonctionnalités

Voici quelques-unes des principales fonctionnalités de l'intégration Brightcove Campaign - Oracle Eloqua :

  • Format des données - Les données de visualisation vidéo sont fournies à un objet de données personnalisé (CDO) dans Oracle Eloqua. Les données répliquées vers Eloqua sont les suivantes :
    • Nom de la vidéo
    • ID de la vidéo
    • ID de compte Brightcove
    • URL de la page de référence
    • ID de lecteur Brightcove
    • % regardé
  • Transfert de données - L'affichage des données datant de plus de 15 minutes est envoyé de Brightcove Campaign à Eloqua quatre fois par heure. Notez que cela signifie que l'affichage des données peut prendre jusqu'à 30 minutes dans Eloqua.

Configuration de la connexion

Avant de configurer l'intégration avec Eloqua, nous vous recommandons d'ouvrir une nouvelle fenêtre de navigateur et de vous déconnecter manuellement de votre compte Eloqua. Cela garantira que lorsque vous configurez l'intégration, vous serez invité à vous connecter à votre compte Eloqua et vous pouvez confirmer que vous utilisez le bon compte.

Remarque : Le processus de connexion ouvre des fenêtres contextuelles dans votre navigateur. Vous devez désactiver tous les bloqueurs de fenêtres contextuelles du navigateur avant de configurer la connexion.

Procédez comme suit pour configurer la connexion entre Brightcove Campaign et Oracle Eloqua.

  1. Connectez-vous à Brightcove Campaign.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis MAP Connections. Une liste des connexions actuelles (le cas échéant) apparaîtra.
  3. Cliquez sur Add New Collection.
  4. Sélectionner Éloqua dans la liste déroulante.
  5. Cliquez sur Se connecter à Eloqua.
  6. Vous serez invité à vous connecter à Eloqua. Saisissez votre nom d' entreprise Eloqua, nom d'utilisateur et mot de passe. Cliquez sur Se connecter.
  7. Si la connexion est réussie, vous serez invité à installer le Connexion vidéo Brightcove pour Eloqua application. Cliquez sur Accepter et installer.
  8. Vous serez invité à activer Eloqua pour le compte. Cliquez sur Activer.
  9. Passez en revue les conditions d'utilisation de l'intégration, puis cliquez sur Je suis d'accord.
  10. Vous serez invité à autoriser Brightcove Video Connect for Eloqua à accéder à votre compte Eloqua. Cliquez sur Se connecter.
  11. Confirmez le nom de votre société et votre nom d'utilisateur et cliquez sur Accep
  12. Confirmez l'affichage des détails de connexion.

La connexion à Eloqua est maintenant configurée et prête à l'emploi. Par défaut, l' état de la connexion sera activé. le Statut de connexion détermine si les données d'engagement vidéo sont envoyées à Eloqua:

  • Activé - Les données sont collectées et envoyées à Eloqua
  • Désactivé - Aucune donnée n'est collectée (l'intégration est désactivée)

Cliquez sur Save pour enregistrer les modifications apportées. Pour supprimer l'intégration, cliquez sur Supprimer.
