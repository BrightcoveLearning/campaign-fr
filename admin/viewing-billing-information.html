Affichage des informations de facturation

{{ page.description }}

Les administrateurs ont la possibilité d'afficher les informations de facturation d'un compte. Notez que pour accéder à la page Facturation, vous devez être un utilisateur avec le rôle Administrateur et vous devez disposer d'un contrat annuel associé à votre compte. Pour accéder à la page Billing :

  1. Connectez-vous à Brightcove Campaign™.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis Facturation.

La page principale de facturation s'affiche. le Compte de facturation le menu déroulant peut être utilisé pour sélectionner un autre compte à gérer.

La page Facturation est organisée en différentes sections qui sont couvertes ci-dessous.

Détails du plan

le DÉTAILS DU PLAN La section répertorie les informations sur votre plan actuel, notamment:

  • Nom du plan
  • Nombre maximal de comptes d'utilisateurs
  • Nombre maximum de vidéos dans le compte
  • Date d'expiration du plan

Infos de facturation

le INFORMATIONS DE FACTURATION affiche les informations de facturation pour le compte actuel. Cliquez sur Responsable de la facturation pour afficher / modifier vos informations de facturation.

Présentation de l'utilisation

le APERÇU DE L'UTILISATION La section affiche des informations sur le nombre de comptes d'utilisateurs qui ont été créés et le nombre de vidéos dans le compte.

Contact

le CONTACT peut être utilisée pour envoyer un e-mail à Brightcove avec des questions sur la facturation.
