Se connecter à Marketo avec Munchkin

{{ page.description }}

Brightcove Campaign peut être configuré pour s'intégrer à Marketo. Lorsque l'intégration est configurée, les données d'engagement vidéo des vidéos publiées à l'aide de Brightcove Campaign sont synchronisées avec Marketo. Une fois dans Marketo, ces données peuvent être utilisées pour la segmentation, la logique de campagne, les rapports, la notation des prospects et la personnalisation des communications.

Remarque : Pour plus d'informations sur la configuration de Brightcove Campaign pour une utilisation avec Marketo à l'aide des API REST, consultez Se connecter à Marketo à l'aide des API REST.

Voici quelques-unes des caractéristiques clés de l'intégration Marketo :

  • Format des données - Les données de visualisation vidéo sont fournies à Marketo sous forme de page vue ou de visite virtuelle sur le Web. L'événement de visite de page Web contient des paramètres de requête qui indiquent le niveau d'engagement de l'utilisateur. Les données répliquées sur Marketo sont les suivantes :
    • Nom de la vidéo
    • ID de la vidéo
    • ID de compte Brightcove
    • URL de la page de référence
    • ID de lecteur Brightcove
    • % regardé
  • Transfert de données - Les données de visualisation sont envoyées immédiatement à Marketo, cependant, certains rapports de la plate-forme Marketo peuvent prendre un certain temps à actualiser
Pour utiliser Brightcove Campaign avec Marketo Munchkin, vous devez avoir un compte Marketo avec l'API Munchkin activée - vous aurez besoin de votre identifiant de compte Munchkin (voir Accéder au Marketo Munchkin au dessous de). Les utilisateurs possédant le rôle Utilisateur Munchkin doivent pouvoir accéder à ces valeurs.

Accéder au Marketo Munchkin

Lorsque vous configurez l'intégration avec Brightcove Campaign, vous serez invité à entrer votre Marketo munchkin. Suivez les étapes ci-dessous pour accéder à cette valeur.

  1. Connectez-vous à votre compte Marketo.
  2. Cliquez sur Admin dans la navigation supérieure.
  3. Dans la navigation de gauche, accédez à Admin > Integration > Munchkin.
  4. Copiez l' ID de compte Munchkin et enregistrez-le dans un endroit sûr. Il sera nécessaire lors de la configuration de l'intégration.

Configuration de la connexion

Procédez comme suit pour configurer la connexion entre Brightcove Campaign et Marketo.

Établir le lien entre Brightcove Campaign et Marketo

  1. Connectez-vous à Brightcove Campaign.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis MAP Connections. Une liste des connexions actuelles (le cas échéant) apparaîtra.
  3. Cliquez sur Add New Collection.
  4. Sélectionnez Marketo dans la liste déroulante.
  5. Sélectionnez le Volume élevé (Munchkin) type d'intégration.
  6. Saisissez votre ID de compte Marketo. Il s'agit de l'ID de compte Munchkin copié depuis Marketo.
  7. Cliquez sur Save.

La connexion à Marketo est maintenant configurée et prête à être utilisée. Par défaut, l' état de la connexion sera activé. L' état de la connexion détermine si les données d'engagement vidéo sont envoyées à Marketo :

  • Activé - Les données sont collectées et seront envoyées à Marketo
  • Désactivé - Aucune donnée n'est collectée (l'intégration est désactivée)

Cliquez sur Save pour enregistrer les modifications apportées. Pour supprimer l'intégration, cliquez sur Supprimer.
