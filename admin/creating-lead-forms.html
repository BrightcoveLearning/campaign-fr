Création de formulaires de prospectes

{{ page.description }}

Les formulaires de prospect peuvent être utilisés pour capturer les informations du spectateur pendant la lecture vidéo. Lors de la création d'un formulaire de prospect, vous devrez coller dans le formulaire de prospect HTML à partir d'une plateforme d'automatisation marketing, puis configurer le calendrier et l'affichage du formulaire. Pour plus d'informations sur la création du formulaire de prospect HTML, consultez le document approprié pour votre plateforme d'automatisation marketing :

  • Oracle Eloqua
  • Marketo
  • HubSpot
  • Salesforce

Création d'un formulaire de clients potentiels

Pour créer un formulaire de clients potentiels, procédez ainsi.

  1. Connectez-vous à Brightcove Campaign.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis Formulaires pour prospects. Une liste de tous les formulaires de prospect (le cas échéant) apparaîtra.
  3. Cliquez sur Create Lead Form.
  4. Collez votre Code du formulaire principal. Il s'agit du formulaire HTML généré à partir de votre logiciel de plateforme d'automatisation marketing.
  5. Sélectionner un Fermer la confirmation temps d'affichage. Cela détermine la durée d'affichage du message de confirmation de fermeture (s'il est entré).
  6. (Optionnel) Entrer le Code d'écran de confirmation. Il s'agit du message qui s'affichera lorsque le formulaire de prospect sera soumis.
  7. Donnez au formulaire un nom descriptif.
  8. Sélectionnez un statut dans Lead Form Status.
    • Actif
    • Inactif
  9. Sélectionnez le formulaire Horaire. Le timing détermine quand le formulaire de prospect sera affiché.
  10. Cochez Require submission to watch video pour exiger que le formulaire de clients potentiels soit envoyé avant de pouvoir regarder la vidéo.
  11. Cocher Always show the lead form even if the lead is already known affichera toujours le formulaire de clients potentiels, même si celui-ci est déjà connu. Si cette option n'est pas cochée, le formulaire prospect n'est pas affiché si le prospect est connu.
  12. (Optionnel) Cliquez sur Aperçu du formulaire pour ouvrir un aperçu du formulaire dans un nouvel onglet du navigateur. La fenêtre d'aperçu permet de prévisualiser le formulaire sur smartphone, tablette ou ordinateurs de bureau.
    Remarque : Si l'aperçu n'apparaît pas, assurez-vous que votre navigateur n'a pas bloqué l'affichage de l'aperçu.
  13. Cliquez sur Save pour enregistrer le formulaire.

Le formulaire de prospect peut être modifié en cliquant sur le nom du formulaire de prospect ou en cochant la case à côté du formulaire de prospect, puis en cliquant sur Éditer.

Affichage des données du formulaire de clients potentiels

Les données soumises sous forme de formulaires prospect peuvent être consultées sur la plateforme d'automatisation marketing appropriée. Les étapes de visualisation des données sont abordées dans les rubriques du formulaire de prospect pour chaque plateforme :

  • Oracle Eloqua
  • Marketo
  • Hub Spot
  • Salesforce
