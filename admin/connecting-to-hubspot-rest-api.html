Se connecter à HubSpot à l'aide des API REST

{{ page.description }}

Brightcove Campaign peut être configuré pour s'intégrer à HubSpot. Lorsque l'intégration est configurée, les données d'engagement vidéo des vidéos publiées à l'aide de Brightcove Campaign sont synchronisées avec HubSpot. Une fois dans HubSpot, ces données peuvent être utilisées pour la segmentation, la logique de campagne, les rapports, la notation des prospects et la personnalisation des communications.

Remarque : Pour plus d'informations sur la configuration de Brightcove Campaign pour une utilisation avec HubSpot à l'aide des API côté client, consultez Connexion à HubSpot à l'aide des API côté client.

Voici quelques-unes des principales caractéristiques de l'intégration Brightcove Campaign - HubSpot :

  • Format de données : l'affichage des données est fourni sous la forme d'un type d'événement de chronologiepersonnalisé. Les données répliquées sur HubSpot sont les suivantes :
    • Nom de la vidéo
    • ID de la vidéo
    • ID de compte Brightcove
    • URL de la page de référence
    • ID de lecteur Brightcove
    • % regardés
  • Transfert de données - L'affichage des données est envoyé à HubSpot à l'aide d'un processus automatisé ; cependant, certains rapports sur la plateforme HubSpot peuvent prendre un certain temps pour actualiser

Configuration de la connexion

Suivez ces étapes pour configurer la connexion à HubSpot.

Remarque : Le processus de connexion ouvre des fenêtres contextuelles dans votre navigateur. Vous devez désactiver tous les bloqueurs de fenêtres contextuelles du navigateur avant de configurer la connexion.

Établissement de la connexion entre Brightcove Campaign et HubSpot

  1. Connectez-vous à Brightcove Campaign.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis MAP Connections. Une liste des connexions actuelles (le cas échéant) apparaîtra.
  3. Cliquez sur Add New Collection.
  4. Sélectionnez HubSpot dans la liste déroulante.
  5. Sélectionnez le Norme (API REST) type d'intégration.
  6. Cliquez sur Se connecter à HubSpot.
  7. Vous serez invité à vous connecter à HubSpot. Saisissez votre adresse e-mail et votre mot de passe. Cliquez sur Se connecter.
  8. Si la connexion est réussie, vous pouvez être invité à autoriser Brightcove Video Connect à accéder à votre compte. Cliquez sur Accorder l'accès.

La connexion à HubSpot est maintenant configurée et prête à l'emploi. Par défaut, l' état de la connexion sera activé. L'état de la connexion détermine si les données d'engagement vidéo sont envoyées à Marketo :

  • Activé - Les données sont collectées et seront envoyées à HubSpot
  • Désactivé - Aucune donnée n'est collectée (l'intégration est désactivée)

Cliquez sur Save pour enregistrer les modifications apportées. Pour supprimer l'intégration, cliquez sur Supprimer.
