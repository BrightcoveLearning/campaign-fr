Configuration des connexions de la plate-forme Marketing Automation

{{ page.description }}

Campagne de Brightcove™ peut être configuré pour s'intégrer aux plates-formes d'automatisation du marketing (MAP) populaires telles que HubSpot, Eloqua et Marketo. Lorsque l'intégration MAP est configurée, les données d'engagement vidéo pour les vidéos publiées à l'aide de Brightcove Campaign sont synchronisées avec la plate-forme d'automatisation marketing sélectionnée.

Remarque : Vous devez être administrateur de compte pour accéder au menu d'administration.
Remarque : Le processus de connexion ouvre des fenêtres contextuelles dans votre navigateur. Vous devez désactiver tous les bloqueurs de fenêtres contextuelles du navigateur avant de configurer la connexion.

Pour configurer une intégration avec une plateforme d'automatisation marketing, procédez comme suit :

  1. Connectez-vous à Brightcove Campaign.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis MAP Connections. Une liste des connexions actuelles (le cas échéant) apparaîtra.
  3. Cliquez sur Add New Collection.
  4. Sélectionnez le type de connexion. Consultez la rubrique appropriée pour plus d'informations sur la connexion à cette plate-forme.
    • Éloqua
    • Marketo (API REST ou Munchkin)
    • HubSpot (API REST ou API côté client)
    • Salesforce
