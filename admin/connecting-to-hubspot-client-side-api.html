Connexion à HubSpot à l'aide des API côté client

{{ page.description }}

Brightcove Campaign peut être configuré pour s'intégrer à HubSpot. Lorsque l'intégration est configurée, les données d'engagement vidéo des vidéos publiées à l'aide de Brightcove Campaign sont synchronisées avec HubSpot. Une fois dans HubSpot, ces données peuvent être utilisées pour la segmentation, la logique de campagne, les rapports, la notation des prospects et la personnalisation des communications.

Remarque : Pour plus d'informations sur la configuration de Brightcove Campaign pour une utilisation avec HubSpot à l'aide des API REST, consultez Se connecter à Marketo à l'aide des API REST.

Voici quelques-unes des principales fonctionnalités de l'intégration Video Cloud - HubSpot :

  • Format des données - L'affichage des données est fourni sous la forme d'un type d'événement chronologique personnalisé. Les données répliquées sur HubSpot sont les suivantes :
    • Nom de la vidéo
    • ID de la vidéo
    • ID de compte (Video Cloud)
    • URL de la page (URL de la page de référence)
    • ID du joueur (lecteur Video Cloud)
    • % regardé (25 %, 50 %, 75 %, 95 %)
  • Transfert de données - La visualisation des données est envoyée à HubSpot à l'aide d'un processus automatisé. Toutefois, certains rapports sur la plateforme HubSpot peuvent prendre du temps à être actualisés
  • La conservation des données - Les données collectées peuvent éventuellement être conservées dans Brightcove Campaign jusqu'à 6 mois

Configuration de la connexion

Suivez ces étapes pour configurer la connexion à HubSpot.

Remarque : Le processus de connexion ouvre des fenêtres contextuelles dans votre navigateur. Vous devez désactiver tous les bloqueurs de fenêtres contextuelles du navigateur avant de configurer la connexion.

Établissement de la connexion entre Brightcove Campaign et HubSpot

  1. Connectez-vous à Brightcove Campaign.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée () puis MAP Connections. Une liste des connexions actuelles (le cas échéant) apparaîtra.
  3. Cliquez sur Add New Collection.
  4. Sélectionnez HubSpot dans la liste déroulante.
  5. Sélectionnez le Volume élevé (API côté client) type d'intégration.
  6. Entrez votre HubSpot identifiant de compte.
  7. Cliquez sur Se connecter à HubSpot.
  8. Vous serez invité à vous connecter à HubSpot. Saisissez votre adresse e-mail et votre mot de passe. Cliquez sur Se connecter.
  9. Si la connexion est réussie, vous pouvez être invité à autoriser Brightcove Video Connect à accéder à votre compte. Cliquez sur Accorder l'accès.

La connexion à HubSpot est maintenant configurée et prête à l'emploi. Par défaut, l' état de la connexion sera activé. L'état de la connexion détermine si les données d'engagement vidéo sont envoyées à Marketo :

  • Activé - Les données sont collectées et seront envoyées à HubSpot
  • Désactivé - Aucune donnée n'est collectée (l'intégration est désactivée)
