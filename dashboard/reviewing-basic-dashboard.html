Consulter le tableau de bord de base

{{ page.description }}

Le tableau de bord de base fournit une vue d'ensemble des vidéos les plus performantes en fonction du pourcentage moyen visionné. Une représentation graphique des impressions vidéo, des démarrages vidéo, du pourcentage moyen de visionnage et de la durée moyenne affichée s'affiche.

Brightcove Campaign affiche un tableau de bord avancé lorsque le champ de métadonnées Type de contenu vidéo pour les vidéos a été activé. Le tableau de bord avancé fournit un score VBS (Video Benchmark Score) pour chaque vidéo qui compare les performances de vos vidéos à celles d'autres vidéos similaires du même type.

Remarque : Brightcove recommande d'activer le champ Type de contenu vidéo, puis d'utiliser le tableau de bord avancé.

Pour afficher la page Tableau de bord, cliquez sur Tableau de bord dans l'en-tête de navigation.

Sélection d’une période

Pour spécifier la période du tableau de bord, vous pouvez cliquer sur une date rapide ou utiliser le contrôle du calendrier.

Utilisation d'une date rapide

Une série de dates rapides se trouvent en haut de la page. Les dates rapides suivantes sont disponibles :

  • Aujourd'hui
  • Hier
  • 7 derniers jours
  • 30 derniers jours
  • Ce mois-ci - Le mois civil à ce jour - par exemple, si aujourd'hui est le 17 mars, ce rapport s'affiche du 1er mars au 17 mars
  • Le mois dernier - le mois civil précédent - par exemple, si aujourd'hui est le 14 février, ce rapport s'affiche du 1er janvier au 31 janvier.

Cliquez sur une date rapide pour mettre à jour le tableau de bord.

Utilisation du champ Calendrier

Vous pouvez également sélectionner des dates de début et de fin en cliquant sur le contrôle du calendrier.

Cliquez sur Appliquer pour mettre à jour le tableau de bord

Fuseau horaire du reporting

Les données du tableau de bord sont affichées à l'aide du fuseau horaire de reporting du compte. Le fuseau horaire est affiché sous le contrôle du calendrier.

Le fuseau horaire du reporting permet de calculer les limites journalières de reporting de données. La modification du fuseau n'affecte que les données à venir. Elle n'est donc pas rétroactive. Si vous modifiez ce paramètre, un creux ou un pic de données peut se produire aux limites du jour où la modification a été apportée. Les mises à jour de ce paramètre ne sont pas nécessairement appliquées immédiatement. Les données peuvent continuer à se baser sur l'ancien fuseau jusqu'à ce que le système prenne en compte la modification.

Le fuseau horaire de reporting peut être configuré dans le cadre des paramètres du compte.

Termes utilisés sur le tableau de bord

Il sera utile de consulter les termes suivants de la vidéo avant de visionner le tableau de bord avancé.

  • Démarrages vidéo : nombre de fois où une vidéo a commencé à jouer
  • Video Impressions : le nombre de fois où une vidéo a été chargée et prête pour interaction.
  • Pourcentage moyen de visualisation - Durée moyenne de vision/durée de la vidéo
  • Temps moyen de visualisation - Temps total regardé / démarré

Vérification des analyses de tableau

Le corps de la page affiche des analyses pour les vidéos les plus performantes. le Vidéos les plus performantes list affichera les cinq meilleures vidéos avec le pourcentage moyen le plus élevé de vues. La liste contiendra :

  • Nom de la vidéo
  • Short description
  • Date de publication
  • Pourcentage moyen de vues

La sélection d'une vidéo dans la liste affichera des analyses de base pour la vidéo. Le survol d'un graphique affichera les valeurs exactes des données.

Cliquez sur Plus d'analyses pour cette vidéo pour ouvrir la page Analytics avec des analyses détaillées pour la vidéo sélectionnée.
